# CoursePilot

CoursePilot is a web-based course registration and waitlist management system.

Students can log in, view available course sections, and submit course registration requests. Each course section has a fixed seat capacity. When a section becomes full, students can join a waiting list and view their current waitlist position.

The system will also allow students to track whether their registration request is pending, approved, rejected, or waitlisted.

## Main Features

- Student login
- Available course and section browsing
- Course registration
- Fixed course-seat capacity
- Waitlist management
- Waitlist-position tracking
- Registration approval and rejection
- Prerequisite checking
- Minimum and maximum credit validation
- Class-schedule conflict detection
- Final-submission blocking for conflicting courses
- Registered-course schedule
- Class time and room information

## Intended Users

- Students
- Academic advisors
- Department administrators
- System administrators

## Proposed Technology Stack

- Frontend: React
- Backend: FastAPI
- Database: PostgreSQL
- API: REST API
- Version Control: Git and GitHub
